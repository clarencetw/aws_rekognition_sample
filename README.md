# Rekognition

## API Document
http://docs.aws.amazon.com/AWSJavaScriptSDK/latest/AWS/Rekognition.html

## 執行 Sample 專案
- 先下載專案，可使用 git clone 或是右上角的 Download Zip
- 進入資料夾下載插件執行 npm install
- 執行程式執行 node main.js

## 安裝 Node.js
- 先到官網下載 INSTALL 檔 https://nodejs.org/en/
